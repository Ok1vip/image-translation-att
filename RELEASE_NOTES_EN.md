# Image Translation ATT 1.1.5

Image Translation ATT is a desktop app that translates text from image and PDF files with no strict limits by combining multiple online translation engines in one simple workflow.

## Recommended download

- `Image-Translation-ATT-Setup.exe` - recommended for most users

## Also included

- `Image-Translation-ATT-Portable.zip` - portable version for manual launch without the installer

## What is new in this base release

- Compact and cleaner interface
- Easier file drop area
- Better visual document cards and status feedback
- Improved OCR processing speed while keeping recognition quality
- Updated developer and support information
