# Image Translation ATT

Image Translation ATT is a Windows desktop app that translates text from image and PDF files in one simple workflow.

It was built for people who want a practical tool without strict limits, confusing setup, or being locked to a single translation service.

You can load one file or many files at once, start the process, and let the app continue working through everything step by step.

## What it does

- Translates text from JPG, JPEG, PNG, BMP, WEBP, and PDF files
- Combines multiple online translation engines in one app
- Keeps the workflow simple for everyday users
- Lets you add multiple files and leave the app to process them in the background
- Includes a compact interface designed for quick document handling
- Includes 14 interface languages
- Supports translation into 25 target languages

## Why it is different

Most tools are limited, complicated, or tied to one platform.

Image Translation ATT was created to give users one place where they can load files, translate them quickly, and keep working without unnecessary restrictions.

In practice, this means you can add a batch of files, start the job, and let the program work through them while you do something else.

The app will continue to receive updates, improvements, and future refinements.
